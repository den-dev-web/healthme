# HealthMe

Мобильное веб-приложение для онлайн‑консультаций с врачами.

## Запуск

Проект статический — достаточно открыть `index.html` в браузере.

## Структура

- `index.html` — главная страница
- `doctor.html` — страница врача
- `src/styles/` — стили
- `src/js/` — скрипты
- `src/assets/` — изображения и иконки
